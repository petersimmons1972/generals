## Observations

### Methodology Insights

**Remote open-access research ceiling**: 85-90/100 accuracy is the MAXIMUM achievable without:
1. Physical archive access (UK/Germany)
2. University library access (paywalled books)
3. German language proficiency
4. Technical PDF extraction solutions

**Institutional sources (Tier 5-6) are appropriate**: When primary sources aren't digitized, credible institutional syntheses (USHMM, IWM) that cite primary documents are the best available option.

**Source chain tracing is critical**: The France 1940 statistics finding demonstrates that widely-cited "verified" statistics can trace back to self-reporting (Tier 6-7) rather than independent verification (Tier 1-2).

**Ideal**: Full reading of primary war diaries, operational records, academic monographs

**Value**: Documenting what's accessible vs. restricted is honest research practice

## Recommendations for Future Missions

### 1. Acknowledge Remote Research Limitations Upfront

Don't promise "thorough standard (full reading)" when primary sources aren't digitized. Instead:
- "Thorough standard (systematic source location + full reading where accessible)"
- "Remote open-access methodology (85-90/100 ceiling)"

### 2. Invest in PDF Extraction Solutions

Current WebFetch limitation blocks access to open-access academic papers. Solutions:
- Alternative extraction tools
- OCR capability for scanned PDFs
- Direct download capability (if user system access available)

### 3. Prioritize Source Chain Tracing

The France 1940 statistics finding demonstrates value of tracing citations to origin:
- Widely-cited ≠ independently verified
- Multiple sources ≠ independent sources (may all cite same origin)
- Check reliability tier of SOURCE, not just number of citations

### 4. Set Realistic Accuracy Expectations

**For remote methodology**:
- 85-90/100: Excellent (institutional sources, cross-verified secondary scholarship)
- 90-95/100: Requires university library access or archive travel
- 95-100/100: Unrealistic (requires comprehensive multilingual primary source access)

**Cost-benefit**: Going from 85 to 90 requires significant resources for 5-point gain. Often not worth it.
