## 2. Propaganda Detection Cross-Reference

| Pattern | Patton | Rommel | Guderian |
|---------|--------|--------|----------|
| **Propaganda source** | Self-generated (real-time) | Constructed by others (3 stages) | Self-generated (post-war memoir) |
| **Primary vector** | Wartime claims, theatrical behavior | Nazi propaganda → British press → Cold War | *Panzer Leader* memoir (1952) |

| **Statistical inflation** | 500k casualties claimed → 163k verified | "Nearly Suez Canal" → 100km from Alexandria | "Father of Blitzkrieg" → one innovator among several |
| **Ethical omission** | Denazification sanitized | Nazi regime relationship omitted | Commissar Order denial vs. own records |
| **Patterns detected** | 10 (4 prev + 6 new) | 6 | 8 |
| **Pre-audit accuracy** | 78/100 | 55-60/100 | N/A (new profile) |
| **Post-audit accuracy** | 90/100 | 85-90/100 | 85-90/100 (estimated) |

### Cross-Cutting Finding: Self-Mythologization

All three pilot generals were self-mythologizers or benefited from mythology, but through different mechanisms:

- **Patton**: Real-time self-promotion (wartime claims, theatrical entries, diary crafting). Source: Daniel III, 2020.
- **Rommel**: Post-hoc mythology by others serving political agendas (Nazi, British, Cold War). Source: Showalter 2006, Reuth 2004.
- **Guderian**: Post-war memoir as legal defense that became history because no alternatives existed. Source: Battistelli ("most remarkable skill was not as a theoretician or commander, but as an author").

**Methodology recommendation for Phase 2:** For every general, ask: "Who built this mythology, and why?" The answer determines the propaganda detection approach.

## 5. Methodology Lessons for Phase 2

### What Worked

1. **Three-journalist model**: Propaganda detection (Orwell), fact-checking (Murrow), humanization (Pyle) provide complementary lenses
2. **Modern scholarship priority**: Post-2000 sources consistently more accurate than pre-1990 popular accounts
3. **Cross-reference requirement**: Sharing findings between researchers caught issues single-researcher investigations would miss
4. **"Who built this mythology?" question**: Most effective propaganda detection heuristic

### What Needs Improvement

1. **Directory structure**: Patton files were created at two different paths. Standardize on `/historical-audit/pilot/{general}/` for Phase 2
2. **Guderian vs. Patton differentiation in Patton proposal**: My Patton proposal still says "(pending Guderian research)" -- needs update now that Guderian is complete
3. **Source counting methodology**: Need consistent format for source quality assessment across all three researchers
4. **Profile format consistency**: All three profiles should follow the same section structure for Phase 2

### Phase 2 Recommendations

1. Apply "three-stage mythology" framework (Nazi/Allied/Cold War) to all German generals
2. Apply "self-mythologizer" framework to all generals with memoirs
3. Prioritize generals with highest propaganda risk (von Manstein next -- HIGH risk)
4. Standardize deliverable format across all researchers
